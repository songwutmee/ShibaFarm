###  Game Concept
ShibaFarm is a game about a Shiba Inu who inherits a massive debt from his father. To pay it all back, you have to work hard every day—chopping trees, planting crops, fishing, and mining. Every cent you earn goes toward paying off the debt collectors until you're finally free.

###  Technical Work
In this project, I focused on making the code clean and easy to work with for the whole team:

- **Used C# Events** to link debt, time, and UI modules. This keeps the systems modular and independent, so they don't break each other.
- **Used the State pattern** for crop growth to handle multi-stage logic efficiently without the performance cost of checking every frame.
- **Built a system with ScriptableObjects** so the team could balance prices, growth time, and item stats instantly without touching any code.
- **Collaborated with artists** to polish "Game Feel" by adding physics-based item magnets and smooth interaction routines.
- **Save/Load System** using JSON to keep track of the farm state, inventory, and how much debt is left to pay.



###  Team & Collaboration
This project was a team effort:
- **Gameplay Programmer (Me):** I was responsible for the technical structure, core gameplay systems, and making sure all the logic worked together.
- **Artists & Designers:** They provided the 3D assets and world design. I worked closely with them to integrate their work and make the game feel responsive and fun to play.
